# Blacklist.txt 사용법
- blacklist.txt는 멜론차트 노래가 아닌 다른 노래가 검색됬을 경우를 대비하여 만들어놨습니다.
- 사용법은 간단합니다.
```
(blacklist.txt)

<스포티파이 음악 이름> * <스포티파이 그 음악의 아티스트 명>

Queencard * (여자)아이들
```
- 이렇게 적으시면 됩니다.
